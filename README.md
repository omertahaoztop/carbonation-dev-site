# Carbonation.dev

Community tooling, libraries and integrations for [Fizzy](https://fizzy.do).

## About

Carbonation.dev is a directory showcasing open source projects built by the community for Fizzy. Browse CLI tools, libraries, integrations, migration tools, and applications.

## Adding Your Project

Built something for Fizzy? We'd love to feature it!

1. Fork this repository
2. Add your project to `index.html` in the appropriate category
3. Submit a pull request

Or email robzolkos@hey.com
